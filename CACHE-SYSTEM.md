# Система кэширования Discord профилей

## Описание

Глобальная система кэширования Discord профилей всех админов для быстрого поиска без повторных запросов к API.

## Как работает

### 1. Загрузка кэша

- При первой загрузке любой страницы `BanGoalsNotifier.js` автоматически загружает все профили админов
- Профили загружаются по 5 одновременно для ускорения
- Промежуточные результаты сохраняются в `localStorage` во время загрузки
- **Автоматическая повторная попытка**: Если какие-то профили не удалось загрузить, система автоматически повторяет попытку через 2 секунды
- Финальный кэш сохраняется в `localStorage` с меткой времени

### 2. Обработка ошибок

- Система отслеживает неудачные загрузки профилей
- После первой попытки загрузки всех профилей, система ждёт 2 секунды
- Затем автоматически повторяет попытку загрузки только для неудачных профилей
- Логирует количество успешно загруженных профилей после повторной попытки
- Если после повторной попытки остались неудачные загрузки, выводит предупреждение в консоль

### 3. Хранение кэша

Кэш хранится в `localStorage` браузера:

- **Ключ**: `discordProfilesCache` - массив профилей
- **Ключ**: `discordProfilesCacheTime` - timestamp создания кэша
- **Ключ**: `discordProfilesCacheLoading` - флаг активной загрузки (временный)

### 4. Время жизни кэша

- **Срок действия**: 24 часа
- **Автоматическое обновление**: Каждый час система проверяет возраст кэша
- **Принудительное обновление**: Если кэш старше 24 часов, он автоматически перезагружается

### 5. Синхронизация между вкладками

- Если одна вкладка уже загружает профили, другие вкладки ждут завершения
- Используется флаг `discordProfilesCacheLoading` для координации
- Таймаут ожидания: 30 секунд

### 6. Использование кэша

```javascript
// Получить кэш из BanGoalsNotifier
const cache = window.banGoalsNotifier.getDiscordProfilesCache();

if (cache.isLoaded) {
    const profiles = cache.profiles;
    // Используйте профили
}
```

## Преимущества

1. **Быстрый поиск**: Все профили загружены заранее, поиск мгновенный
2. **Экономия трафика**: Профили загружаются один раз в сутки
3. **Работает офлайн**: После загрузки кэш доступен без интернета
4. **Синхронизация**: Кэш общий для всех вкладок браузера
5. **Автообновление**: Кэш автоматически обновляется раз в сутки
6. **Надёжность**: Автоматическая повторная попытка при ошибках загрузки

## Страницы, использующие кэш

- `discord-search.html` - поиск по Discord username
- Все остальные страницы загружают кэш в фоне для будущего использования

## Отладка

Откройте консоль браузера (F12) и смотрите логи:

```
[Discord Cache] Loaded from localStorage: 50 profiles
[Discord Cache] Cache age: 5 hours
[Discord Cache] Loaded 45/50 profiles...
[Discord Cache] Retrying 5 failed profiles...
[Discord Cache] Retry successful: +4 profiles (49/50 total)
[Discord Cache] 1 profiles failed to load after retry
[Discord Cache] ✓ Finished loading and saved to localStorage: 49 profiles
[Discord Cache] Cache will be valid for 24 hours
```

## Очистка кэша

Для принудительной очистки кэша выполните в консоли:

```javascript
localStorage.removeItem('discordProfilesCache');
localStorage.removeItem('discordProfilesCacheTime');
localStorage.removeItem('discordProfilesCacheLoading');
location.reload();
```

## Технические детали

- **Файл**: `fear-protection-v2/js/BanGoalsNotifier.js`
- **Класс**: `BanGoalsNotifier`
- **Метод загрузки**: `loadDiscordProfilesInBackground()`
- **Метод проверки**: `checkCacheAge()`
- **Интервал проверки**: 1 час (3600000 мс)
- **Размер батча**: 5 профилей одновременно
- **Задержка перед повторной попыткой**: 2 секунды
- **Максимум попыток**: 2 (основная + повторная)
