# Fear Protection V2 - Система авторизации

## Что нового в V2?

### 🔐 Система авторизации
- Вход через Fear токен
- Автоматическая проверка валидности токена
- Отображение профиля администратора в header
- Защита всех страниц от неавторизованного доступа

### 👤 Профиль администратора
- Аватар и имя из Fear API
- Dropdown меню с:
  - Ссылкой на профиль Fear
  - Кнопкой выхода
- Автоматическое обновление данных профиля

### 🎨 Новый дизайн страницы входа
- Современный glassmorphism дизайн
- Анимированный фон
- Валидация токена в реальном времени
- Красивые сообщения об ошибках

## Как запустить

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
node server.js
```

3. Откройте браузер:
```
http://localhost:3002
```

## Как получить токен?

Токен можно получить:
1. В Discord сервере Fear Protection
2. Через Fear API (для разработчиков)

## API Endpoints

### Проверка токена
```
GET https://api.fearproject.ru/profile/me
Headers:
  Authorization: Bearer {token}
```

## Безопасность

- Токен хранится в localStorage
- Автоматическая проверка валидности при каждой загрузке страницы
- Редирект на login при невалидном токене
- Защита всех страниц через AuthManager
