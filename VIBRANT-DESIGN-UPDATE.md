## Что изменилось

### 🎨 Новая цветовая схема

#### Фон
- **Старый**: `#050505` (почти чёрный)
- **Новый**: `#0f0f1e` (тёмно-синий с фиолетовым оттенком)
- **Эффект**: Многослойный градиентный фон с анимацией

#### Стекло (Glass)
- **Старый**: `rgba(255, 255, 255, 0.05)` (едва заметное)
- **Новый**: `rgba(255, 255, 255, 0.12-0.20)` (более заметное и яркое)

#### Границы (Borders)
- **Старый**: `rgba(255, 255, 255, 0.08)` (белые, едва заметные)
- **Новый**: `rgba(167, 139, 250, 0.2-0.6)` (фиолетовые, яркие)

#### Акценты
- **Фиолетовый**: `#a78bfa` (более яркий)
- **Голубой**: `#22d3ee` (более яркий)
- **Розовый**: `#f472b6` (новый)
- **Оранжевый**: `#fb923c` (новый)

### ✨ Новые эффекты

#### Анимированный фон
```css
/* Многослойный градиент с анимацией */
background-image: 
    radial-gradient(at 20% 30%, rgba(167, 139, 250, 0.15) 0%, transparent 50%),
    radial-gradient(at 80% 70%, rgba(34, 211, 238, 0.15) 0%, transparent 50%),
    radial-gradient(at 50% 50%, rgba(244, 114, 182, 0.1) 0%, transparent 50%);

/* Анимация градиента */
animation: gradient-shift 15s ease infinite;
```

#### Кнопки с блеском
```css
/* Эффект блеска на кнопках */
.btn-primary::after {
    background: linear-gradient(45deg, transparent, rgba(255, 255, 255, 0.3), transparent);
    animation: shine 3s infinite;
}
```

#### Карточки с свечением
```css
.card:hover {
    box-shadow: 
        0 16px 48px rgba(167, 139, 250, 0.3), 
        0 0 60px rgba(167, 139, 250, 0.2);
}
```

#### Inputs с фокусом
```css
input:focus {
    box-shadow: 
        0 0 0 4px rgba(167, 139, 250, 0.2), 
        0 0 30px rgba(167, 139, 250, 0.3);
}
```

### ОБЯЗАТЕЛЬНО: Очистите кэш!

**Способ 1: Жёсткая перезагрузка**
```
Windows/Linux: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

**Способ 2: DevTools**
```
1. F12 → Network
2. Правый клик на Reload
3. "Empty Cache and Hard Reload"
```

**Способ 3: Полная очистка**
```
1. Ctrl + Shift + Delete
2. "Cached images and files"
3. "Clear data"
```
