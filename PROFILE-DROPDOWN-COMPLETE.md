## Функционал выпадающего меню

### Структура меню:
```
┌─────────────────────────────────────┐
│  [Avatar]  Santa2555555             │
│            АДМИН+                   │
├─────────────────────────────────────┤
│  👤  Мой профиль                    │
│  📊  Админ панель                   │
│  🔗  Профиль на сайте               │
├─────────────────────────────────────┤
│  🚪  Выйти                          │
└─────────────────────────────────────┘
```

### Пункты меню:

1. **Мой профиль** 
   - Переход на `profile.html?steamid={steamId}`
   - Показывает детальную информацию о пользователе

2. **Админ панель**
   - Переход на `admin-panel.html`
   - Главная страница админ-панели

3. **Профиль на сайте**
   - Открывает `https://fearprotection.ru/profile` в новой вкладке
   - Внешняя ссылка на основной сайт

4. **Выйти**
   - Удаляет токен из localStorage
   - Перенаправляет на `login.html`
   - Запрашивает подтверждение

### Управление меню:

**Открытие:**
- Клик по кнопке профиля в header

**Закрытие:**
- Клик вне меню
- Нажатие клавиши ESC
- Клик по любому пункту меню

## Технические детали

### ProfileDropdown.js

**Основные методы:**
- `init()` - Инициализация при загрузке
- `createDropdown()` - Создание HTML структуры меню
- `attachEventListeners()` - Привязка обработчиков событий
- `toggle()` - Переключение состояния меню
- `open()` - Открытие меню
- `close()` - Закрытие меню
- `loadProfile()` - Загрузка данных профиля
- `updateUI()` - Обновление интерфейса
- `logout()` - Выход из системы

**Автоматическая загрузка профиля:**
```javascript
// При загрузке страницы автоматически:
// 1. Декодирует JWT токен
// 2. Получает Steam ID
// 3. Загружает профиль через API
// 4. Обновляет аватар, имя и роль
```

### RoleAccessControl.js

**Основные методы:**
- `checkAccess()` - Проверка доступа пользователя
- `hasAccess(profile)` - Проверка роли профиля
- `redirectToLogin()` - Перенаправление на логин
- `showAccessDenied(profile)` - Показ страницы отказа

**Проверка ролей:**
```javascript
// Проверяет по двум полям:
// 1. profile.adminGroup.group_name (STAFF, ADMIN, и т.д.)
// 2. profile.adminGroup.group_display_name (Стафф, Админ, и т.д.)
```
