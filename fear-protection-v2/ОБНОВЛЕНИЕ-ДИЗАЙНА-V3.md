## 🔧 Если изменения не видны

### Вариант 1: Жёсткая перезагрузка
```
Ctrl + Shift + R (или Cmd + Shift + R на Mac)
```

### Вариант 2: Режим инкогнито
```
Chrome/Edge: Ctrl + Shift + N
Firefox: Ctrl + Shift + P
```

### Вариант 3: Полная очистка кэша
```
1. Ctrl + Shift + Delete
2. Выберите "Кэш" или "Cached images and files"
3. Нажмите "Удалить данные"
```

### Вариант 4: Перезапустите браузер
Полностью закройте браузер и откройте заново.

### Вариант 5: Другой браузер
Попробуйте Chrome, Firefox или Edge.
