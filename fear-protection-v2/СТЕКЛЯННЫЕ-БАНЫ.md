## 🎯 Технические детали

### **Использованные эффекты:**
- `backdrop-filter: blur(12px-40px)` - размытие фона
- `background: var(--glass-*)` - стеклянный фон
- `border: 2px solid var(--border-*)` - фиолетовые границы
- `box-shadow: 0 0 20px rgba(167,139,250,0.3)` - свечение
- `transition: all 0.3s ease` - плавные переходы

### **CSS переменные:**
```css
--glass-ultra-light: rgba(255, 255, 255, 0.08)
--glass-light: rgba(255, 255, 255, 0.12)
--glass-medium: rgba(255, 255, 255, 0.15)
--glass-strong: rgba(255, 255, 255, 0.20)

--border-subtle: rgba(167, 139, 250, 0.2)
--border-light: rgba(167, 139, 250, 0.3)
--border-medium: rgba(167, 139, 250, 0.4)
--border-strong: rgba(167, 139, 250, 0.6)

--blur-sm: 12px
--blur-md: 20px
--blur-lg: 32px
--blur-xl: 40px
```
