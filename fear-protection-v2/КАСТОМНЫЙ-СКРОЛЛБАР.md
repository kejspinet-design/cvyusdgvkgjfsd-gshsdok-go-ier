# 📜 Кастомный маленький скроллбар

## ✅ Что сделано

Добавлен кастомный маленький скроллбар с красным градиентом для страницы логина и модального окна.

---

## 🎨 Дизайн скроллбара

### **Для модального окна:**
- **Ширина**: 6px (очень тонкий)
- **Трек**: Тёмный полупрозрачный (rgba(0,0,0,0.2))
- **Ползунок**: Красный градиент (#b91c1c → #dc2626)
- **Hover**: Светлее + свечение
- **Закруглённые углы**: 10px

### **Для всей страницы:**
- **Ширина**: 8px (тонкий)
- **Трек**: Тёмный полупрозрачный (rgba(0,0,0,0.3))
- **Ползунок**: Красный градиент (#b91c1c → #dc2626)
- **Hover**: Светлее + свечение
- **Закруглённые углы**: 10px

---

## 🎯 Особенности

### **Градиент:**
```css
background: linear-gradient(180deg, #b91c1c 0%, #dc2626 100%);
```
- Тёмно-красный сверху
- Светло-красный снизу
- Плавный переход

### **Hover эффект:**
```css
background: linear-gradient(180deg, #dc2626 0%, #ef4444 100%);
box-shadow: 0 0 10px rgba(220, 38, 38, 0.6);
```
- Становится светлее
- Появляется красное свечение
- Плавная анимация (0.3s)

### **Поддержка браузеров:**
- ✅ **Chrome/Edge** - ::-webkit-scrollbar
- ✅ **Firefox** - scrollbar-width: thin + scrollbar-color
- ✅ **Safari** - ::-webkit-scrollbar

## 💻 Код

### **Для модального окна:**
```css
.modal-content::-webkit-scrollbar {
    width: 6px;
}

.modal-content::-webkit-scrollbar-track {
    background: rgba(0, 0, 0, 0.2);
    border-radius: 10px;
}

.modal-content::-webkit-scrollbar-thumb {
    background: linear-gradient(180deg, #b91c1c 0%, #dc2626 100%);
    border-radius: 10px;
    transition: all 0.3s ease;
}

.modal-content::-webkit-scrollbar-thumb:hover {
    background: linear-gradient(180deg, #dc2626 0%, #ef4444 100%);
    box-shadow: 0 0 10px rgba(220, 38, 38, 0.5);
}

/* Firefox */
.modal-content {
    scrollbar-width: thin;
    scrollbar-color: #dc2626 rgba(0, 0, 0, 0.2);
}
```

### **Для всей страницы:**
```css
::-webkit-scrollbar {
    width: 8px;
    height: 8px;
}

::-webkit-scrollbar-track {
    background: rgba(0, 0, 0, 0.3);
}

::-webkit-scrollbar-thumb {
    background: linear-gradient(180deg, #b91c1c 0%, #dc2626 100%);
    border-radius: 10px;
    transition: all 0.3s ease;
}

::-webkit-scrollbar-thumb:hover {
    background: linear-gradient(180deg, #dc2626 0%, #ef4444 100%);
    box-shadow: 0 0 10px rgba(220, 38, 38, 0.6);
}

/* Firefox */
* {
    scrollbar-width: thin;
    scrollbar-color: #dc2626 rgba(0, 0, 0, 0.3);
}
```

## 🚀 Как проверить

1. **Очистите кэш браузера**:
   ```
   Ctrl + Shift + R
   ```

2. **Откройте страницу логина**:
   ```
   fear-protection-v2/login.html
   ```

3. **Откройте модальное окно** (нажмите "Инструкция")

4. **Проверьте скроллбар**:
   - Прокрутите модальное окно вниз
   - Наведите на скроллбар
   - Проверьте свечение
