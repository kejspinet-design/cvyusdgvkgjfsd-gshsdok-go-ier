# Пустое состояние и порядок уведомлений

## Дата: 8 мая 2026

## Изменения

### 1. Улучшенное пустое состояние для страницы серверов

**Файл**: `fear-protection-v2/js/AntiCheatPage.js`

**Что изменено**:
- Пустое состояние теперь отображается по центру экрана (fixed positioning)
- Добавлена анимация плавающего смайлика (float animation)
- Увеличен размер смайлика до 80px
- Улучшена типографика:
  - Заголовок "Пусто": 28px, жирный (font-weight: 700)
  - Подзаголовок: 16px, полупрозрачный (rgba(255, 255, 255, 0.6))

**Когда показывается**:
- Нет доступных серверов
- Нет игроков онлайн
- Нет игроков с публичными профилями

**Пример**:
```
        😴 (плавающий смайлик)
        Пусто
    Нет доступных серверов
```

### 2. Изменён порядок уведомлений

**Файл**: `fear-protection-v2/js/BanGoalsNotifier.js`

**Что изменено**:
- Уведомления теперь добавляются в **начало** контейнера (insertBefore)
- Старые уведомления остаются **сверху**
- Новые уведомления появляются **снизу**

**До**:
```javascript
this.notificationContainer.appendChild(notification);
// Новые уведомления добавлялись в конец (снизу)
```

**После**:
```javascript
this.notificationContainer.insertBefore(notification, this.notificationContainer.firstChild);
// Новые уведомления добавляются в начало (сверху)
```

**Визуальный порядок** (снизу вверх):
```
┌─────────────────────────┐
│ Старое уведомление #1   │ ← Самое старое (сверху)
├─────────────────────────┤
│ Старое уведомление #2   │
├─────────────────────────┤
│ Новое уведомление #3    │ ← Самое новое (снизу)
└─────────────────────────┘
```

## Технические детали

### Анимация пустого состояния

```css
@keyframes float {
    0%, 100% { transform: translateY(0px); }
    50% { transform: translateY(-10px); }
}
```

- Длительность: 3 секунды
- Тип: ease-in-out
- Бесконечный цикл (infinite)
- Смайлик плавно поднимается на 10px и опускается обратно

### Позиционирование уведомлений

```css
position: fixed;
bottom: 20px;
right: 20px;
display: flex;
flex-direction: column;
gap: 10px;
```

- Контейнер уведомлений закреплён в правом нижнем углу
- Уведомления располагаются вертикально (column)
- Отступ между уведомлениями: 10px

## Тестирование

### Пустое состояние
1. Откройте `fear-protection-v2/anticheat.html`
2. Если нет серверов/игроков, увидите:
   - Плавающий смайлик 😴 по центру экрана
   - Заголовок "Пусто"
   - Сообщение о причине

### Порядок уведомлений
1. Откройте любую страницу с уведомлениями
2. Создайте несколько уведомлений (например, прогресс банов)
3. Проверьте порядок:
   - Первое уведомление должно быть **сверху**
   - Последнее уведомление должно быть **снизу**
