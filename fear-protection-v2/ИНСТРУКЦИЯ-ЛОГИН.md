## 📋 Содержание инструкции

### **Что такое Fear токен?**
Объяснение, что токен (access_token) — это ключ доступа для администраторов.

### **Как получить токен (7 шагов):**
1. Перейти на сайт FearProject
2. Войти в свой аккаунт
3. Открыть инструменты разработчика (F12)
4. Перейти в раздел Cookies (Application → Cookies → fearproject.ru)
5. Найти access_token в списке cookies
6. Скопировать весь текст из колонки "Value"
7. Вставить токен на странице входа

### **Визуальная инструкция:**
Показан путь к токену в DevTools:
```
F12 → Application (вкладка)
└─ Storage (левая панель)
   └─ Cookies
      └─ https://fearproject.ru
         └─ access_token ← скопировать значение
```

### **Предупреждения о безопасности:**
- ⚠️ Не передавать токен никому
- ⚠️ Не публиковать в открытых каналах
- ⚠️ Сменить пароль на FearProject при компрометации
- ⚠️ Токен действителен пока вы авторизованы

### **Решение проблем:**
- Токен не работает → проверить полноту копирования из "Value"
- Не вижу access_token → убедиться что авторизованы на FearProject
- Доступ запрещён → проверить права (модератор или выше)
- Токен истёк → перезайти на FearProject

## 🎯 Функциональность

### **Открытие модального окна:**
```javascript
// При клике на "Инструкция"
showInstructionsBtn.addEventListener('click', (e) => {
    e.preventDefault();
    instructionsModal.classList.add('show');
});
```

### **Закрытие модального окна:**
1. **По кнопке ×**:
   ```javascript
   closeModalBtn.addEventListener('click', () => {
       instructionsModal.classList.remove('show');
   });
   ```

2. **По клику вне окна**:
   ```javascript
   instructionsModal.addEventListener('click', (e) => {
       if (e.target === instructionsModal) {
           instructionsModal.classList.remove('show');
       }
   });
   ```

3. **По клавише Escape**:
   ```javascript
   document.addEventListener('keydown', (e) => {
       if (e.key === 'Escape' && instructionsModal.classList.contains('show')) {
           instructionsModal.classList.remove('show');
       }
   });
   ```
