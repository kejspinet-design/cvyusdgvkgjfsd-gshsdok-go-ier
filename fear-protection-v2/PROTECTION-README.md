## ⚙️ Настройка (опционально)

### Отключить для разработки

В файле `AntiDevTools.js` в начале добавить:

```javascript
// Отключить на localhost
if (window.location.hostname === 'localhost') {
    return;
}
```

### Усилить защиту

Раскомментировать в `AntiDevTools.js`:

```javascript
// Обфускация текста
window.addEventListener('load', obfuscateText);

// Очистка storage
localStorage.clear();
sessionStorage.clear();
```

## ⚠️ Важно

- Защита работает только на клиенте
- Опытный хакер может обойти
- Для полной защиты нужна серверная валидация
- Не храните секреты в JavaScript коде
