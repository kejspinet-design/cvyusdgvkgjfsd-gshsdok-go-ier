# Настройка логирования авторизаций в Google Таблицу

## Шаг 1: Создание Google Таблицы

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу
3. Назовите её **"Fear Protection - Auth Logs"**

## Шаг 2: Настройка Apps Script

1. В Google Таблице откройте: **Расширения → Apps Script**
2. Удалите весь код по умолчанию
3. Скопируйте содержимое файла `google-sheets-logger.gs`
4. Вставьте код в редактор Apps Script
5. Нажмите **Сохранить** (иконка дискеты)

## Шаг 3: Развертывание веб-приложения

1. Нажмите **Развернуть → Новое развертывание**
2. Нажмите на иконку шестерёнки рядом с "Выберите тип"
3. Выберите **"Веб-приложение"**
4. Заполните настройки:
   - **Описание**: `Fear Protection Auth Logger`
   - **Запуск от имени**: `Меня (ваш email)`
   - **У кого есть доступ**: `Все`
5. Нажмите **Развернуть**
6. **Скопируйте URL веб-приложения** (он будет выглядеть так: `https://script.google.com/macros/s/XXXXXXXX/exec`)

## Шаг 4: Обновление кода сайта

1. Откройте файл `fear-protection-v2/js/AuthManager.js`
2. Найдите строку:
   ```javascript
   this.GOOGLE_SHEETS_WEBHOOK = 'https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec';
   ```
3. Замените `YOUR_DEPLOYMENT_ID` на ваш URL из шага 3
4. Сохраните файл

## Шаг 5: Тестирование

1. Откройте сайт Fear Protection
2. Авторизуйтесь с токеном
3. Проверьте Google Таблицу - должна появиться новая строка с данными:
   - Дата и время
   - Steam ID
   - Никнейм
   - Роль
   - Токен (сокращённый)
   - Главный админ (ДА/Нет)

## Безопасность

- Токены сохраняются в сокращённом виде (первые 50 символов)
- Доступ к таблице имеет только владелец Google аккаунта
- Webhook URL можно изменить в любой момент через Apps Script

## Отладка

Если логирование не работает:

1. Откройте консоль браузера (F12)
2. Проверьте логи: `[AuthManager] Logging auth to Google Sheets`
3. Проверьте Apps Script:
   - Откройте **Выполнение** в меню слева
   - Посмотрите логи выполнения
4. Убедитесь, что URL webhook правильный

## Примечания

- Логирование происходит при каждой успешной авторизации
- Если Google Sheets недоступен, авторизация всё равно пройдёт успешно
- Главный админ (Steam ID: `76561199524780327`) имеет полный доступ ко всему функционалу
