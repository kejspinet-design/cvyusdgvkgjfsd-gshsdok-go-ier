# Система норм наказаний по ролям

## Обзор

Система автоматически отслеживает выполнение норм наказаний для каждой роли администрации. Нормы отображаются на странице "Цели банов" (`ban-goals.html`).

---

## Нормы по ролям

### Мл. Модератор
- **Еженедельная норма**: 20 банов
- **Ежемесячная норма**: 100 банов

### Модератор
- **Еженедельная норма**: 25 банов
- **Ежемесячная норма**: 130 банов

### Ст. Модератор
- **Еженедельная норма**: нет
- **Ежемесячная норма**: 80 банов

### Ст. Админ
- **Еженедельная норма**: нет
- **Ежемесячная норма**: 50 банов

---

## Как это работает

### 1. Определение роли
Система автоматически определяет роль пользователя из профиля:
```javascript
const role = profile.adminGroup?.group_name || profile.role;
```

### 2. Подсчёт банов
Система считает баны за два периода:
- **Неделя**: последние 7 дней
- **Месяц**: последние 30 дней

### 3. Отображение прогресса
На странице отображается:
- Текущее количество банов / Норма
- Прогресс-бар с процентом выполнения
- Количество оставшихся банов до выполнения нормы

## Статусы выполнения

### Активна (active)
- Норма ещё не выполнена
- Прогресс-бар: синий градиент
- Цвет статуса: зелёный

### Выполнена (completed)
- Норма выполнена на 100% или больше
- Прогресс-бар: заполнен полностью
- Цвет статуса: синий

## Технические детали

### Конфигурация норм
```javascript
const ROLE_QUOTAS = {
    'Мл. Модератор': { weekly: 20, monthly: 100 },
    'Модератор': { weekly: 25, monthly: 130 },
    'Ст. Модератор': { weekly: 0, monthly: 80 },
    'Ст. Админ': { weekly: 0, monthly: 50 }
};
```

### API эндпоинты
- `GET /api/player?steamid={steamId}` - получение профиля и роли
- `GET /api/admin/punishments?type=0` - получение списка банов

### Обновление данных
- Автоматическое обновление каждые 30 секунд
- Обновление при загрузке страницы
- Обновление после создания новой цели

## Добавление новой роли

Чтобы добавить норму для новой роли:

1. Откройте `ban-goals.html`
2. Найдите объект `ROLE_QUOTAS`
3. Добавьте новую роль:

```javascript
const ROLE_QUOTAS = {
    'Мл. Модератор': { weekly: 20, monthly: 100 },
    'Модератор': { weekly: 25, monthly: 130 },
    'Ст. Модератор': { weekly: 0, monthly: 80 },
    'Ст. Админ': { weekly: 0, monthly: 50 },
    'Новая Роль': { weekly: 15, monthly: 75 }  // ← Новая роль
};
```

**Примечание**: Если у роли нет еженедельной нормы, установите `weekly: 0`.

---

## Интеграция с целями

Система норм работает независимо от пользовательских целей:
- **Нормы по ролям** - автоматические, основаны на роли
- **Пользовательские цели** - создаются вручную, любой период

Обе системы отображаются на одной странице и работают параллельно.
