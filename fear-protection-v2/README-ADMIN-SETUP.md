## 📋 Что нужно сделать

### Шаг 1: Настроить Google Таблицу (опционально)

Если хотите логировать авторизации:

1. **Создайте Google Таблицу**
   - Откройте https://sheets.google.com
   - Создайте новую таблицу
   - Назовите "Fear Protection - Auth Logs"

2. **Настройте Apps Script**
   - В таблице: Расширения → Apps Script
   - Скопируйте код из файла `google-sheets-logger.gs`
   - Вставьте в редактор
   - Сохраните

3. **Разверните веб-приложение**
   - Нажмите "Развернуть → Новое развертывание"
   - Выберите тип: "Веб-приложение"
   - Запуск от имени: "Меня"
   - У кого есть доступ: "Все"
   - Нажмите "Развернуть"
   - **Скопируйте URL**

4. **Обновите код**
   - Откройте `js/AuthManager.js`
   - Найдите строку:
     ```javascript
     this.GOOGLE_SHEETS_WEBHOOK = 'https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec';
     ```
   - Замените `YOUR_DEPLOYMENT_ID` на ваш URL
   - Сохраните

### Шаг 2: Тестирование

1. **Откройте сайт**
   ```
   http://localhost:3002/admin-access-test.html
   ```

2. **Установите токен**
   - Нажмите кнопку "Установить токен"

3. **Проверьте доступ**
   - Перейдите на любую страницу
   - Откройте консоль (F12)
   - Должно быть сообщение:
     ```
     [AuthManager] ✅ MAIN ADMIN ACCESS - Full access granted
     ```

4. **Проверьте логирование** (если настроили Google Sheets)
   - Откройте Google Таблицу
   - Должна появиться новая строка с вашими данными
