## Формат токена

Все API запросы используют единый формат:

```javascript
const token = localStorage.getItem('fearToken');

const response = await fetch('/api/endpoint', {
    method: 'GET',
    headers: {
        'Authorization': `Bearer ${token}`,
        'Accept': 'application/json'
    }
});
```

## Проверка токена

### Автоматическая проверка:

1. **При загрузке страницы:**
   - `RoleAccessControl.js` проверяет наличие токена
   - Если токена нет → редирект на `login.html`

2. **При каждом API запросе:**
   - Токен передаётся в заголовке `Authorization`
   - Сервер проверяет валидность токена
   - Если токен невалиден → API возвращает 401

### Ручная проверка:

Откройте консоль браузера (F12) и выполните:

```javascript
const token = localStorage.getItem('fearToken');
console.log('Token:', token ? '✓ Установлен' : '❌ Не установлен');

if (token) {
    const payload = JSON.parse(atob(token.split('.')[1]));
    console.log('Steam ID:', payload.client_id);
    console.log('Expires:', new Date(payload.exp * 1000).toLocaleString('ru-RU'));
}
```
