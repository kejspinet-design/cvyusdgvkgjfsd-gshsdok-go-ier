# Исправление наложения виджетов

## Дата: 8 мая 2026

## Проблема

Виджеты "Цели банов" и "Нормы наказаний" накладывались друг на друга в правом нижнем углу.

### Причина
Оба виджета использовали одинаковое позиционирование:
```css
position: fixed;
bottom: 20px;
right: 20px;
```

## Решение

Изменено позиционирование всех виджетов - теперь они располагаются **сверху вниз** в правом верхнем углу.

### Новое расположение (сверху вниз):

1. **Нормы наказаний** - `top: 80px` (z-index: 9998)
2. **Цели банов** - `top: 220px` (z-index: 9997)
3. **Discord кэш** - `top: 380px` (z-index: 9996, динамическая позиция)
4. **Уведомления** - `top: 380px` (z-index: 9995)

## Расположение виджетов

```
Правый верхний угол:

┌─────────────────────────────┐
│  📊 Нормы наказаний         │ ← top: 80px
│  Норма: 100 | Сделано: 45   │
└─────────────────────────────┘
              ↓ 20px отступ
┌─────────────────────────────┐
│  🎯 Цели банов              │ ← top: 220px
│  45/100 банов (45%)         │
└─────────────────────────────┘
              ↓ 20px отступ
┌─────────────────────────────┐
│  💬 Discord кэш             │ ← top: 380px (динамически)
│  Загрузка профилей...       │
└─────────────────────────────┘
              ↓ 10px отступ
┌─────────────────────────────┐
│  🔔 Уведомления             │ ← top: 380px+
│  Прогресс обновлён          │
└─────────────────────────────┘
```

## Изменения в коде

### PunishmentNormsWidget.js
```javascript
// Было:
bottom: 340px;
z-index: 9998;

// Стало:
top: 80px;
z-index: 9998;
```

### BanGoalsNotifier.js

#### Виджет целей банов:
```javascript
// Было:
bottom: 20px;
z-index: 9999;

// Стало:
top: 220px;
z-index: 9997;
```

#### Уведомление Discord кэша:
```javascript
// Было:
bottom: 20px;
z-index: 9997;

// Стало:
top: 80px; (динамически 80px/240px/380px)
z-index: 9996;
```

#### Контейнер уведомлений:
```javascript
// Было:
bottom: 20px;
z-index: 10000;

// Стало:
top: 380px;
z-index: 9995;
```

## Динамическое позиционирование

Уведомление Discord кэша автоматически подстраивается:
- Если есть оба виджета → `top: 380px`
- Если только виджет норм → `top: 240px`
- Если нет виджетов → `top: 80px`
